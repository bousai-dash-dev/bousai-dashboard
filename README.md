# 防災ダッシュボード

Twitterをやめても災害時の情報収集に困らないための、個人用PWAプロトタイプ。

## 何ができるか

- **地点タブ**: 現在地(GPS) / 登録地点(自宅・実家など) / 期限付きの一時滞在地(旅行先)を切り替え。全国どこでも同じダッシュボードが出る
- **津波・地震の速報**: P2P地震情報のAPI+WebSocketでリアルタイム受信。震度5弱以上・津波予報はブラウザ通知
- **指定緊急避難場所マップ**: 国土地理院のデータを災害種別(津波/地震/洪水など)で切替表示。各避難場所からGoogleマップ経路リンク
- **Xの生の情報**: Yahoo!リアルタイム検索への定型クエリボタン(「〇〇市 避難」「〇〇市 停電」など)。**Xアカウント不要・ログイン不要**
- **外部リンク**: 重ねるハザードマップ / 通れた道マップ / 気象庁 / ウェザーニュース

## 起動方法(ローカル)

```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1
```

→ ブラウザで http://localhost:8123/ を開く。

Node.jsやPythonは不要(PowerShell製の簡易サーバ)。

## データソース

| 情報 | ソース | 方式 |
|---|---|---|
| 地震・津波 | P2P地震情報 API (api.p2pquake.net) | REST + WebSocket |
| 避難場所 | 国土地理院 指定緊急避難場所 (skhb01〜08) | GeoJSONタイル (z=10) |
| 住所検索・逆ジオコーディング | 国土地理院 msearch / mreversegeocoder | REST |
| 地図タイル | 国土地理院 淡色地図 | ラスタタイル |
| Xの投稿 | Yahoo!リアルタイム検索 | 定型クエリのディープリンク |

## 今後の課題(次のステップ)

1. **iPhoneから使う**: GPS・通知・PWAインストールにはHTTPSが必要。GitHub Pages / Cloudflare Pages に置くだけで対応可能(全ファイル静的なのでそのままデプロイできる)
2. **閉じていても届くプッシュ通知**: 現状はページを開いている間のみ。Web Push化には小さなサーバ(またはNERV防災アプリとの併用)が必要
3. **気象警報(大雨・暴風など)**: 気象庁JSONの追加

## 免責

個人用プロトタイプ。避難の判断は必ず気象庁・自治体の公式発表に従うこと。
